# FreePremium - Premium Video Streaming Platform

Welcome to FreePremium, a state-of-the-art video streaming website built step-by-step.

## Project Structure
- `index.html` - Core structure and landing experience
- `styles.css` - Custom premium styling (modern dark mode, glassmorphism, HSL color system)
- `app.js` - Dynamic interactions, video players, and state management
